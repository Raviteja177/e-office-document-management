# E-Office Document Management System

## Description
This is a web-based E-Office / Document Management System developed using PHP and MySQL.
It allows users to securely log in, upload documents, view records, and monitor activities using a dashboard with charts.

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- Chart.js
- Apache (XAMPP)

## Features
- Secure Login System
- Role-Based Access
- Document Upload & View
- Dashboard Analytics
- User Management

## How to Run the Project
1. Install XAMPP
2. Copy project folder into `htdocs`
3. Start Apache & MySQL
4. Import database from `/database/myproject_db.sql`
5. Open browser:
   http://localhost/myproject/
